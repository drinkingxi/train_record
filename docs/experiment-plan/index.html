基础数据

目前最强信号是 v4.x verbatim/no-colon。后续不应重复 v3 数据版本实验，除非某个 slice 结论需要复验。

置信度：high

开源 ATC / ATCOSIM

ATCOSIM 对英文、较长时长和真实来源 slice 有优势，是第一优先迁移到 v4.3 多 epoch 的分支。

置信度：medium-high

App 降噪

App denoise 是稳定的增强候选；多于 1 epoch 后表现更合理，因此保留 45h 多 epoch 验证。

置信度：medium

综合混合

Balanced fullmix 可作为综合混合基准，但变量太多；需要同时做受控配比实验来拆解贡献。

置信度：medium

TTS

旧 TTS 只有 1 epoch，不能直接排除。需要补 TTS-only 和 TTS-in-combo 的多 epoch 实验后再判断。

置信度：low-medium

长音频

长音频 concat 目前集中在 167，不在 47/178 重复。判断依据应是长时长 slice、early-EOS 诊断和 full-WER 是否退化。

置信度：low-medium
