当前判断

基础数据

目前最强信号是 v4.x verbatim/no-colon。后续不应重复 v3 数据版本实验，除非某个 slice 结论需要复验。

置信度：high

baseline

exp29 v4.3 baseline 已评估到 5 epoch：full WER 13.6621 / CER 13.0874 / SER 65.6943；现沿用同一 exp29 output_dir 从 checkpoint-23085 续跑到 8 epoch，用于确认 baseline 是否仍继续收益。

置信度：high

开源 ATC / ATCOSIM

ATCOSIM 对英文、较长时长和真实来源 slice 有优势，是第一优先迁移到 v4.3 多 epoch 的分支。

置信度：medium-high

App 降噪

exp66 app denoise 90h 已完成 3 epoch：full WER 13.3763 / CER 12.7943 / SER 64.7302，相比 exp29 5ep 基线 full、英文、长音频和 YouTube slice 均有改善，但略弱于 exp67 DeepFilter；保留为有效增强候选，继续对照 exp51 45h 与后续 15h 递减。

置信度：medium-high

综合混合

混合数据已包含通用 ASR、降噪、DeepFilter、VAD、open ATC 和 TTS；但 fullmix 变量太多，只作为综合基准，不直接当最终配方。

置信度：medium

通用数据

exp65 general 100h 已完成 3 epoch：full WER 13.4777，收敛稳定且略优于当前 baseline/TTS10h 可比结果，但英文、YouTube/B站和长音频 slice 仍弱；general 可保留为组合 regularizer，不单独作为主方向晋级。

置信度：medium

deepfilter

exp67 DeepFilter 60h 已完成 3 epoch：full WER 13.2894 / CER 12.7750 / SER 64.5424，优于 exp65 general100h，但长音频、英文、YouTube/B站 slice 仍弱；DeepFilter 可保留为高剂量组合候选，不是 long-audio/eos 主解。

置信度：medium

TTS

TTS10h exp63 到 epoch4 full WER 13.4075，优于当前 exp64 TTS22h epoch2 13.8738；TTS22h 在 167 的重复 run 已停，只保留 47 上正在跑的有效进度。TTS 是否进入最终配方需要看 exp63/64 完成后的 numbers/callsigns、mixed、英文 slice。

置信度：medium

长音频

会话 019eee5a 和当前 mini/rich-slice 证据基本确认：提升原始长音频片段占比/上采样是更优先的训练修正方向。由于 ATCOSIM 信号最强且当前急需找到最佳数据组合，后续优先跑 ATCOSIM + 原始长片段上采样（exp80）；base-only 上采样（exp79）降为低优先级机制验证。

置信度：medium-high

atcosim_lead

exp50 ATCOSIM 是当前最强方向：epoch4 full WER 11.1440，duration_long_audio_ge8s WER 16.5065，均显著优于 exp29 baseline 和其他增强；后续应围绕 ATCOSIM 长训与受控组合展开。

置信度：high

long_mini

exp73/74 已完成 mini 对照：exp73 全量 WER 16.8633，>=6s WER 22.2367 优于 baseline 23.5371，>=8s WER 22.6158 也略优于 baseline 22.8883；exp74 最好全量 WER 17.3460，弱于 exp61/73。full-derived rich eval 正在 167 跑，完成后只用于确认长音频 slice 和 early-EOS，不改变“上采样优先、concat 降级”的当前计划。

置信度：medium

training_policy

后续有希望的方向不再只跑 3-5 epoch：强候选默认设置 12 epoch，并启用 load_best_model_at_end + early_stopping_patience=3；每 epoch full eval 并派生 slice，按 full WER、目标 slice 和 early-EOS 诊断共同选 checkpoint。

置信度：high

augmentation_direction_summary

截至 2026-06-25：ATCOSIM 是主收益方向；后续组合实验默认基于 ATCOSIM。原始长片段上采样是长音频/EOS 专项优先方向，先跑 exp80（ATCOSIM+long upsample）寻找最佳组合，exp79 仅作低优先级机制验证；DeepFilter/AppDenoise/TTS10/general20 作为小剂量候选。

置信度：medium-high
