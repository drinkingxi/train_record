当前判断

基础数据

目前最强信号是 v4.x verbatim/no-colon。后续不应重复 v3 数据版本实验，除非某个 slice 结论需要复验。

置信度：high

baseline

exp29 v4.3 baseline 已完成 8 epoch；训练期 best full WER 13.3011 @ epoch7，epoch8 回退到 13.3606。固定 generate full/rich eval 下 exp29 fixed full WER 10.3151，是当前已补评估 run 里最稳的基线之一。

置信度：high

开源 ATC / ATCOSIM

ATCOSIM 对英文、较长时长和真实来源 slice 有优势，是第一优先迁移到 v4.3 多 epoch 的分支。

置信度：medium-high

App 降噪

exp66 app denoise 90h 已完成 3 epoch：full WER 13.3763 / CER 12.7943 / SER 64.7302，相比 exp29 当前续跑到 epoch6 的 full WER 13.3743 基本持平，中文略好但英文/长音频不占优；已在 167 GPU2 启动 exp30 app15h 递减剂量验证，继续等待 exp51 45h 完成。

置信度：medium-high

综合混合

混合数据已包含通用 ASR、降噪、DeepFilter、VAD、open ATC 和 TTS；但 fullmix 变量太多，只作为综合基准，不直接当最终配方。

置信度：medium

通用数据

exp65 general 100h 已完成 3 epoch：full WER 13.4777，收敛稳定且略优于当前 baseline/TTS10h 可比结果，但英文、YouTube/B站和长音频 slice 仍弱；general 可保留为组合 regularizer，不单独作为主方向晋级。

置信度：medium

deepfilter

exp67 DeepFilter 60h 已完成 3 epoch：full WER 13.2894 / CER 12.7750 / SER 64.5424，是非 ATCOSIM 单类增强里较强方向；但英文、YouTube/B站和长音频 slice 仍弱。已在 167 GPU3 启动 exp32 DeepFilter15h 递减剂量验证，不作为 long-audio/eos 主解。

置信度：medium

TTS

TTS10h exp63 epoch5 full WER 13.2777；TTS22h exp64 epoch5 best 在 epoch4，full WER 13.1655，epoch5 轻微回退到 13.1714。TTS 有一定收益，但仍弱于 ATCOSIM/ATCOSIM+long，只作为小剂量候选。

置信度：medium

长音频

exp79/80/81 的训练期 full eval 和 fixed generate slice 均未证明长音频上采样是主收益方向：exp79 fixed full WER 11.3111，exp80/81 fixed full WER 21.0145/20.6282 且英文、mixed、numbers/callsigns 不稳；后续不再扩大原长音频/长音频混合，只保留诊断观察。

置信度：medium-high

atcosim_lead

exp50 ATCOSIM 训练期 best 仍是 epoch4 full WER 11.1440，epoch5/6 回退到约 11.39；但固定 generate full/rich eval 下 exp50 fixed full WER 19.8995，暴露插入/重复/EOS 风险。ATCOSIM 仍是训练期强信号，但必须和 generate 诊断分开看。

置信度：high

long_mini

exp73/74 已完成 mini 对照：exp73 全量 WER 16.8633，>=6s WER 22.2367 优于 baseline 23.5371，>=8s WER 22.6158 也略优于 baseline 22.8883；exp74 最好全量 WER 17.3460，弱于 exp61/73。full-derived rich eval 正在 167 跑，完成后只用于确认长音频 slice 和 early-EOS，不改变“上采样优先、concat 降级”的当前计划。

置信度：medium

training_policy

后续有希望的方向不再只跑 3-5 epoch：强候选默认设置 12 epoch，并启用 load_best_model_at_end + early_stopping_patience=3；每 epoch full eval 并派生 slice，按 full WER、目标 slice 和 early-EOS 诊断共同选 checkpoint。

置信度：high

augmentation_direction_summary

截至 2026-06-29 17:06：基础 v4.3 已原地更新到 16:34 版本；后续新增/待启动 run 必须带 latest v4.3 fingerprint，派生数据需从该版本重新 materialize。训练期 full WER 与固定 generate full/rich eval 分开记录；ATCOSIM 训练期收益仍强，但 fixed generate 暴露重复/EOS 风险。
