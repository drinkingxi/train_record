ATC ASR Train Record

This GitHub Pages site hosts static training report assets for WxPusher image embeds.

Expected image paths are updated in place, for example core-slice.png, supplement-slice.png, full-wer.png, and progress.png.

Usage

Point WxPusher HTML <img> tags to the generated PNG files under this site.

Notes

This repository is intentionally public and should contain only derived static report images.
